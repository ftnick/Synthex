Version 2.0.0 - Saturday, 17 February 2024
  • New Homepages for different grouped assets:
    • Roblox Assets now have a dedicated homepage.
    • Python Tools also have their own homepage for better accessibility.
  • Introduced "Roblox Music Downloader" Python tool.
    • Users can download music from Roblox using the hydra.io API.
    • Integrated with a user-friendly GUI application.
  • Added "Youtube Asset Downloader" Python tool.
    • Allows users to download either audio or videos from YouTube with a specified URL.
  • Updated the website format for better organization.
  • Enhanced user experience with a cleaner and more intuitive design.
  • Important Update:
    • As of Saturday, 17 February 2024, all assets will be organized and sorted into their respective home pages accessible through the Asset Pages section.
  • Website Changes:
    • Reorganized links for easy navigation.
    • Improved accessibility with separate homepages for different asset categories.
  • Bug Fixes:
    • Addressed minor bugs and improved overall website performance.
  • Miscellaneous:
    • Updated external libraries and dependencies for better compatibility.
    • Incorporated AOS library for smoother page animations.
    • Continuously striving for better user experience and functionality.
